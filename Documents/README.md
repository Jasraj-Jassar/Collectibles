# Documents Folder

Drop your folders and files here. The app will display them automatically.

## Supported formats

- **PDF** - Rendered inline with the browser's PDF viewer
- **Images** - JPG, PNG, GIF, etc.
- **Text files** - TXT, MD (markdown rendered), and other text formats

## Example structure

```
Documents/
├── Research/
│   ├── notes.md
│   └── diagram.png
├── Reports/
│   └── quarterly.pdf
└── Misc/
    └── todo.txt
```

Each top-level folder appears as a tile on the main screen.
